# Veritell Website

Marketing website for Veritell - Clarity for Caregivers.

A simple, static website showcasing Veritell's mission to help caregivers manage their loved ones' health information.

## Structure

- `index.html` - Main HTML page
- `styles.css` - Stylesheet
- `script.js` - JavaScript for interactions

## Development

Simply open `index.html` in a browser or serve it with any static file server.
